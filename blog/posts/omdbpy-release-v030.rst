`Omdb.py v0.3.0 <https://github.com/dgilland/omdb.py/tree/v0.3.0>`_ has been released.

It was a feature release that added support for OMDb API's new ``type`` parameter.

Changes
-------

Features
++++++++

- Added support for OMDb API's new ``type`` query parameter to all API functions.
- Added ``search_movie()`` to limit searching to movies.
- Added ``search_episode()`` to limit searching to episodes.
- Added ``search_series()`` to limit searching to series.

Bugfixes
++++++++

None
