`Pydash v3.1.0 <https://github.com/dgilland/pydash/tree/v3.1.0>`_ has been released.

It was a minor feature release with some new functions and extended callback style support.

Changes
-------

Features
++++++++

- Added ``fill``.
- Added ``in_range``.
- Added ``matches_property``.
- Added ``spread``.
- Added ``start_case``.
- Made callbacks support ``matches_property`` style as ``[key, value]`` or ``(key, value)``.
- Made callbacks support shallow ``property`` style callbacks as ``[key]`` or ``(key,)``.

+++++++++

None
