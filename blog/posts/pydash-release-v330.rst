`Pydash v3.3.0 <https://github.com/dgilland/pydash/tree/v3.3.0>`_ has been released.

It was feature release that brought it on par with `Lodash v3.10.0 <https://github.com/lodash/lodash/tree/3.10.0>`_.

Changes
-------

Features
++++++++

- Add ``ceil``.
- Add ``defaults_deep``.
- Add ``floor``.
- Add ``get``.
- Add ``gt``.
- Add ``gte``.
- Add ``is_iterable``.
- Add ``lt``.
- Add ``lte``.
- Add ``map_keys``.
- Add ``method``.
- Add ``method_of``.
- Add ``mod_args``.
- Add ``set_``.
- Add ``unzip_with``.
- Add ``zip_with``.
- Make ``add`` support adding two numbers if passed in positionally.
- Make ``get`` main definition and ``get_path`` its alias.
- Make ``set_`` main definition and ``deep_set`` its alias.

Bug Fixes
+++++++++

None
